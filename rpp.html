LAMPIRAN 4: RINGKASAN MATERI AJAR PENDUKUNG

RINGKASAN MATERI PEMBELAJARAN (ADMINISTRASI INFRASTRUKTUR JARINGAN)

1. SEGMENTASI JARINGAN (VLAN & TRUNKING)
VLAN memecah broadcast domain fisik menjadi domain logis terpisah pada switch layer 2 guna tingkatkan efisiensi bandwidth dan aspek keamanan (Disiplin SOP). Port trunking (IEEE 802.1Q) menyalurkan banyak ID VLAN dalam satu link inter-switch. Inter-VLAN routing menjembatani komunikasi antar segmen VLAN lewat router (sub-interface/router-on-a-stick) atau switch layer 3.

2. REDUNDANSI KONEKSI (ROUTING STATIS & FAILOVER)
Routing statis dikonfigurasi manual dengan mendefinisikan network tujuan dan next-hop. Guna cegah terputusnya koneksi akibat putusnya link fisik, diimplementasikan Floating Static Route dengan memanfaatkan nilai Administrative Distance (AD). Jalur utama memiliki AD lebih kecil (misal 1), jalur cadangan diberi AD lebih tinggi (misal 5).

3. ROUTING DINAMIS SKALA INDUSTRI (OSPF)
Open Shortest Path First (OSPF) memakai algoritma Dijkstra untuk kalkulasi rute terdekat berdasar metrik cost link. OSPF membagi jaringan menjadi beberapa area (Area 0 / Backbone wajib ada) guna lokalisasi update LSA (Link State Advertisement), kurangi beban CPU/RAM router pada infrastruktur skala menengah-besar.

4. KEAMANAN JARINGAN (FIREWALL NAT & ACL)
NAT (Network Address Translation) menerjemahkan IP Privat ke IP Publik. PAT (NAT Overload) menghemat IP publik dengan memetakan port unik untuk tiap koneksi internal. Access Control List (ACL) menyaring paket data secara sekuensial berdasarkan alamat IP asal/tujuan, protokol, serta nomor port tcp/udp untuk penegakan kebijakan keamanan digital.

5. PROXY & TRAFFIC MONITORING
Web proxy bertindak sebagai perantara koneksi client, berfungsi memblokir situs web tidak aman/tidak pantas (Budi Luhur). Simple Network Management Protocol (SNMP) memantau status kesehatan hardware (CPU load, memory, bandwidth) secara tersentral ke Network Monitoring Tool, didukung Syslog untuk logging aktivitas event error sistem secara terpusat.
