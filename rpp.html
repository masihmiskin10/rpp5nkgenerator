LAMPIRAN 4: RINGKASAN MATERI PEMBELAJARAN

RINGKASAN MATERI PEMBELAJARAN KEAMANAN JARINGAN 1. KEBIJAKAN PENGGUNAAN JARINGAN (KPJ) & ANCAMAN: KPJ (Network Usage Policy) mengatur hak akses, tanggung jawab, dan batasan penggunaan infrastruktur jaringan. Ancaman meliputi Sniffing, Spoofing, Brute Force, Denial of Service (DoS), dan Man-in-the-Middle (MitM). 2. STATEFUL FIREWALL (IPTABLES): Bekerja memeriksa status paket (ESTABLISHED, RELATED, NEW, INVALID). Rule dasar membatasi input dengan default policy DROP. Syntax utama: `iptables -A INPUT -p tcp --dport 22 -j ACCEPT`. 3. PORT KNOCKING: Mekanisme keamanan dengan cara menyembunyikan port aktif di balik firewall. Port hanya akan terbuka jika klien mengirimkan paket data ke urutan port tertentu secara berurutan sesuai konfigurasi pada file `/etc/knockd.conf`. 4. SERVER AUTENTIKASI (AAA & RADIUS): Authentication (siapa user), Authorization (apa haknya), Accounting (apa aktivitasnya). FreeRADIUS mensentralisasi otentikasi user perangkat jaringan kabel, nirkabel, maupun server lokal. 5. INTRUSION DETECTION SYSTEM (IDS) SNORT: Sistem monitoring trafik real-time yang menggunakan signatures untuk mengenali anomali paket data. Aturan ditulis dalam file `/etc/snort/rules/local.rules`. 6. HARDENING SERVICES & SSL/TLS: Langkah mengamankan OS dan aplikasi. SSH Hardening: matikan root login, ubah port 22 ke port kustom, gunakan SSH Key. SSL/TLS: mengenkripsi komunikasi web server (HTTP -> HTTPS) menggunakan sertifikat digital. 7. KRIPTOGRAFI: Seni mengamankan pesan. Kriptografi Simetris (AES, DES) menggunakan satu kunci untuk enkripsi/dekripsi. Kriptografi Asimetris (RSA) menggunakan pasangan public key dan private key. 8. VIRTUAL PRIVATE NETWORK (VPN): Teknologi untuk membangun jalur komunikasi terenkripsi (tunnel) melalui jaringan publik. Protokol populer: OpenVPN dan IPsec yang menjamin kerahasiaan (confidentiality) dan integritas data remote.
